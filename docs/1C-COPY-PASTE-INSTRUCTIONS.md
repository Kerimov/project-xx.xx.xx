# Куда копировать код 1С

Скопируйте **оба** файла в указанные места. Код в файлах уже полный — копируйте файл целиком.

---

## 1. Файл `1C-UH-Example-Code.bsl` → Модуль обработки

| Шаг | Действие |
|-----|----------|
| 1 | Конфигуратор 1С → **Обработки** → **ОбработкаПорталЕЦОФ_Документы** → откройте **Модуль объекта**. |
| 2 | В модуле: **Ctrl+A** → **Delete**. |
| 3 | Откройте `docs/1C-UH-Example-Code.bsl` → **Ctrl+A** → **Ctrl+C**. |
| 4 | В модуле объекта: **Ctrl+V** → **Ctrl+S**. |

**В HTTP-сервисе** для шаблонов укажите обработчик **ОбработкаПорталЕЦОФ_Документы** и процедуры:
- POST `/documents` → **ОбработатьСозданиеДокумента**
- GET `/health` → **ОбработатьHealthCheck**
- GET `/nsi/delta` → **ОбработатьПолучениеДельтыНСИ**

---

## 2. Файл `1C-NSI-Delta-Procedure-ONLY.bsl` → Модуль HTTP-сервиса

| Шаг | Действие |
|-----|----------|
| 1 | Конфигуратор 1С → **HTTP-сервисы** → **ПорталЕЦОФ** → откройте **Модуль** (не «Шаблоны URL»). |
| 2 | В модуле: **Ctrl+A** → **Delete**. |
| 3 | Откройте `docs/1C-NSI-Delta-Procedure-ONLY.bsl` → **Ctrl+A** → **Ctrl+C**. |
| 4 | В модуле HTTP-сервиса: **Ctrl+V** → **Ctrl+S**. |

**В HTTP-сервисе** для шаблонов укажите обработчик **модуль HTTP-сервиса** и функции:
- GET `/nsi/delta` → **ОбработатьПолучениеДельтыНСИ**
- GET `/health` → **ОбработатьHealthCheck**

*Либо* можно вести `/health` и `/nsi/delta` с обработки (тогда для них указываете обработку и процедуры из пункта 1; модуль HTTP-сервиса всё равно лучше заполнить кодом из файла 2 на случай переключения).

---

## Кратко

| Куда в 1С | Какой файл |
|-----------|------------|
| Модуль объекта обработки **ОбработкаПорталЕЦОФ_Документы** | `1C-UH-Example-Code.bsl` |
| Модуль HTTP-сервиса **ПорталЕЦОФ** | `1C-NSI-Delta-Procedure-ONLY.bsl` |

После вставки обоих файлов: **Сохранить конфигурацию** и **Опубликовать** HTTP-сервис.
