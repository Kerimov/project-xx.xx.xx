## Структура файлов

```
portal/src/
├── config/
│   └── documentGroups.ts          # Конфигурация групп документов
├── components/
│   └── forms/
│       └── BaseDocumentForm.tsx   # Базовый компонент формы
├── pages/
│   ├── documents/
│   │   ├── ReceiptRightsPage.tsx      # Поступление прав
│   │   ├── InvoiceFromSupplierPage.tsx # Счет от поставщика
│   │   ├── PowerOfAttorneyPage.tsx    # Доверенность
│   │   └── AdvanceReportPage.tsx      # Авансовый отчет
│   ├── CreateDocumentPage.tsx         # Роутинг по типам документов
│   ├── CreateReceiptServicesPage.tsx  # Поступление услуг
│   └── SelectDocumentTypePage.tsx     # Выбор типа документа
```

## Навигация

Страница выбора типа документа (`/documents/new`) организована как Collapse с разделами:
- Каждый раздел содержит подразделы
- В каждом подразделе список документов
- Реализованные документы помечены зеленым тегом "Реализовано"

## Базовый компонент формы

`BaseDocumentForm` - универсальный компонент для всех форм документов:
- Заголовок карточки
- Кнопки "Записать" и "Заморозить"
- Единообразный стиль

## Статусы реализации

- ✅ **Реализовано** - форма создана и работает
- ⏳ **В планах** - тип определен, форма не создана
