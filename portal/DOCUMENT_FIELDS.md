# Поля карточек документов (точные копии 1С УХ)

## Поступление товаров: Накладная, УПД

### Основные реквизиты:
- **Накладная, УПД №** (обязательное) - номер документа
- **от:** (обязательное) - дата документа с временем
- **Номер:** - дополнительный номер документа
- **Контрагент** (обязательное) - поиск по ИНН или наименованию
- **Договор** - выбор из списка договоров
- **Счет на оплату** - выбор из списка счетов
- **Есть расхождения** (чекбокс) - флаг наличия расхождений
- **Организация** (обязательное) - выбор организации
- **Склад** (обязательное) - выбор склада (только для товаров)
- **Расчеты:** - текстовое поле с информацией о сроках и счетах расчетов
- **Грузоотправитель и грузополучатель** (ссылка) - открывает форму для заполнения
- **НДС сверху. НДС включен в стоимость** (ссылка) - настройки НДС
- **Оригинал: получен** (чекбокс)

### Табличная часть "Товары":
- **N** - порядковый номер строки
- **Номенклатура** - наименование товара
- **Количество** - количество товара
- **Ед.** - единица измерения
- **Цена** - цена за единицу
- **Сумма** - сумма без НДС (автоматический расчет)
- **% НДС** - процент НДС (0%, 10%, 20%)
- **НДС** - сумма НДС (автоматический расчет)
- **Всего** - итоговая сумма с НДС (автоматический расчет)
- **Счет учета** - счет бухгалтерского учета
- **Страна происхождения** - страна происхождения товара

### Итоги и настройки:
- **УПД** (чекбокс) - флаг универсального передаточного документа
- **Счет-фактура:** - выбор: "Не требуется" / "Требуется"
- **Всего:** - общая сумма без НДС
- **НДС (в т.ч.):** - общая сумма НДС

### Кнопки действий:
- Добавить
- Подбор
- Изменить
- Добавить по штрихкоду

## Поступление услуг: Акт, УПД

### Основные реквизиты:
- **Акт, УПД №** (обязательное) - номер документа
- **от:** (обязательное) - дата документа с временем
- **Номер:** - дополнительный номер документа
- **Контрагент** (обязательное) - поиск по ИНН или наименованию
- **Договор** - выбор из списка договоров
- **Счет на оплату** - выбор из списка счетов
- **Организация** (обязательное) - выбор организации
- **Период оказания услуг:** - текстовое поле (например: "январь 2026")
- **Дата начала:** - дата начала оказания услуг
- **Дата окончания:** - дата окончания оказания услуг
- **Расчеты:** - текстовое поле с информацией о сроках и счетах расчетов
- **Оригинал: получен** (чекбокс)

**Отличия от товаров:**
- ❌ Нет поля "Склад" (не применимо для услуг)
- ❌ Нет поля "Есть расхождения" (не применимо для услуг)
- ❌ Нет ссылок "Грузоотправитель и грузополучатель" и "НДС сверху"
- ✅ Добавлены поля периода оказания услуг (даты начала и окончания)

### Табличная часть "Услуги":
- **N** - порядковый номер строки
- **Услуга** - наименование услуги
- **Количество** - количество услуг
- **Ед.** - единица измерения (обычно "усл")
- **Цена** - цена за единицу
- **Сумма** - сумма без НДС (автоматический расчет)
- **% НДС** - процент НДС (0%, 10%, 20%)
- **НДС** - сумма НДС (автоматический расчет)
- **Всего** - итоговая сумма с НДС (автоматический расчет)
- **Счет учета** - счет бухгалтерского учета

**Отличия от товаров:**
- ❌ Нет колонки "Страна происхождения" (не применимо для услуг)

### Итоги и настройки:
- **УПД** (чекбокс) - флаг универсального передаточного документа
- **Счет-фактура:** - выбор: "Не требуется" / "Требуется"
- **Всего:** - общая сумма без НДС
- **НДС (в т.ч.):** - общая сумма НДС

### Кнопки действий:
- Добавить
- Подбор
- Изменить
- ❌ Нет кнопки "Добавить по штрихкоду" (не применимо для услуг)

## Ключевые различия между карточками

| Поле | Поступление товаров | Поступление услуг |
|------|---------------------|-------------------|
| Склад | ✅ Обязательное | ❌ Отсутствует |
| Есть расхождения | ✅ Есть | ❌ Отсутствует |
| Грузоотправитель/грузополучатель | ✅ Есть ссылка | ❌ Отсутствует |
| НДС сверху/включен | ✅ Есть ссылка | ❌ Отсутствует |
| Период оказания услуг | ❌ Отсутствует | ✅ Есть |
| Дата начала/окончания | ❌ Отсутствует | ✅ Есть |
| Страна происхождения (в таблице) | ✅ Есть колонка | ❌ Отсутствует |
| Добавить по штрихкоду | ✅ Есть кнопка | ❌ Отсутствует |
